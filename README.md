# GFG-Scrapper

A GeeksforGeeks Scrapper to track your friends history. 

Requirements:- 
1. A stable internet connection
2. Python3 <br />
Python libraries - 
    1. requests <br />
    2. html5lib <br />
    3. bs4
    4. pandas
> Install these individually using pip <br />
  or run "pip install -r requirements.txt" in command line
